## A simple File sharing system using WebRTC

### Installation

- Clone the Repo
```
git clone https://github.com/devanshkansagra/FileShare
```

- Configure `.env`
```
mv .env.example .env
```

- Install the packages 
```
npm install
```

- Start Client
```
npm run dev
```

Optional: If wanted to run on network
```
npx vite --host
```

- Start Signalling Server
```
npm run start
```
